## 🚀 **Quick Deploy Commands**

### **1. Instant Setup (2 minutes)**
```bash
cd url-scraper-package
npm run setup
npm start
```

### **2. Test the API**
```bash
npm run test:api
curl http://localhost:3000/health
```

### **3. Production Deploy**
```bash
# Docker
npm run docker:build && npm run docker:run

# PM2
npm run pm2:start
```
